# LITEFLIX

## Challenge

### Stack

- Yarn workspace
- Express
- Typescript
- React
- Vite
- Tailwind
- PostCss

### Run the project

#### Install dependencies

`yarn`

#### Start the API (http://localhost:5005 | https://jeremias-gp-liteflix-api.onrender.com)
`yarn start-api`

Update `workspaces/web/.env` to use local or deployed api

#### Start WEB (http://localhost:5173)
`yarn start-web`
